Alexa BrowserHelp

Full voice control of your Chrome browser via your Amazon Echo

  • The Idea

    Browserhelp was created with the idea to make the web more accessible for the impaired, and allows anyone to control their Chrome browser by voice interaction with an Amazon Echo. To start using the skill, enable the Alexa skill and Chrome extension. The source code can be viewed here on Github

  • BrowserHelp Alexa Skill

    Enable the skill through the skills marketplace, and select Account Linking to match your skill with the Chrome extension

  • BrowserHelp Chrome Extension

    Add the Chrome extension to your browser, and follow the instructions
